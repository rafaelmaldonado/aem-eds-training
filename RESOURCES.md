# Resources

High-trust sources for AEM Edge Delivery Services. Prefer official `aem.live` docs and Adobe GitHub repos over third-party blogs.

## Primary (official)
- **Developer tutorial** — https://www.aem.live/developer/tutorial — 15-min project setup from boilerplate. _(Trust: high, Adobe official)_
- **Anatomy of blocks / markup, sections, blocks** — https://www.aem.live/developer/markup-sections-blocks — how a Doc table becomes `div.block` DOM + `decorate()`. _(Trust: high)_
- **Block Collection** — https://www.aem.live/developer/block-collection — production-grade reference blocks. _(Trust: high)_
- **Boilerplate repo** — https://github.com/adobe/aem-boilerplate — the starting template.
- **Block Collection code** — https://github.com/adobe/aem-block-collection/tree/main/blocks — real block source.
- **Exploring blocks** — https://www.aem.live/docs/exploring-blocks
- **CLI reference** — https://www.aem.live/developer/cli-reference

## Tools
- **AEM CLI** — `npm i -g @adobe/aem-cli`, then `aem up`.
- **AEM Sidekick** — Chrome extension for preview/publish across environments.
- **Document Authoring (DA)** — https://da.live — content source used in the tutorial.

## Community (wisdom)
- **AEM Live Discord** — https://discord.gg/aem-live — active, Adobe-adjacent, best place to ask real questions.
